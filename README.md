# try-go

## Практика 1. Типы данных
Реализовал простейшую структуру пользователя и несколько методов для работы с ней. Ответы на вопросы:
1. RenameByValue не меняет исходного юзера поттому, что получает на вход копию данных, а не ссылку на них.
2. AddTag работает из-за наличия zero value
3. Причина в разной природе структуры данных этих сущностей.
4. slice может быть nil, указывает на данные, а не хранит их (фактически, ближе к коллекции, чем к JS-массиву)
